# 🗑️ MunInstaller

**Professional Windows Uninstaller with Microsoft Store Apps Support**

![Version](https://img.shields.io/badge/version-1.0-blue)
![Platform](https://img.shields.io/badge/platform-Windows-lightgrey)

---

## ✨ Features

- ✅ **Desktop Applications** - Uninstall traditional programs
- ✅ **Microsoft Store Apps** - Remove Store apps  
- ✅ **Force Uninstall** - Remove stuck/corrupted programs
- ✅ **Batch Uninstall** - Uninstall multiple programs at once
- ✅ **Dark Mode** - Eye-friendly dark theme
- ✅ **Search & Filter** - Quickly find programs
- ✅ **Modern UI** - Clean, professional interface

---

## ⌨️ Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| F5 / Ctrl+R | Refresh list |
| Delete / Ctrl+U | Uninstall |
| Ctrl+Shift+U | Force Uninstall |
| Ctrl+Shift+B | Batch Uninstall |
| Ctrl+D | Dark Mode |
| Ctrl+F | Search |
| F1 | Help |

---

## 🛠️ Technology

- .NET 8.0
- Windows Forms
- C#
- PowerShell (for Store apps)

---

## 📋 Requirements

- Windows 10/11
- .NET 8.0 Runtime
- Admin rights

---

## 🚀 How to Build

1. Clone or download this repository
2. Open `MunInstaller.sln` in Visual Studio 2022
3. Build in Release mode (Ctrl+Shift+B)
4. Find EXE in `bin/Release/net8.0-windows/`
