# 🗑️ MunInstaller

**Professional Windows Uninstaller with Microsoft Store Apps Support**

## Features
- ✅ Desktop Applications
- ✅ Microsoft Store Apps
- ✅ Force Uninstall
- ✅ Batch Uninstall
- ✅ Dark Mode
- ✅ Modern UI

## Keyboard Shortcuts
- F5 - Refresh
- Ctrl+D - Dark Mode
- Ctrl+U - Uninstall
- Delete - Uninstall selected

## System Requirements
- Windows 10/11
- .NET 8.0 Runtime
- Admin rights required

## Version
1.0 - Initial Release
